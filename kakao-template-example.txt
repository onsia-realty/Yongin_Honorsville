# 카카오톡 알림톡 템플릿 예시
# NHN Toast 콘솔에서 이 템플릿을 등록해야 합니다.

템플릿 코드: CUSTOMER_REGISTRATION
템플릿명: 관심고객 등록 알림

템플릿 내용:
----------------------------------------
[클러스터용인 경남아너스빌] 관심고객 등록 알림

새로운 관심고객이 등록되었습니다.

▶ 고객 정보
• 성함: #{name}
• 연락처: #{phone}
• 등록시간: #{timestamp}

빠른 연락 부탁드립니다.

▶ 문의: 1668-5257
----------------------------------------

변수 설명:
- #{name}: 고객명
- #{phone}: 고객 연락처
- #{timestamp}: 등록 시간

버튼 추가 가능:
- URL 버튼: 관리 페이지로 이동
- 전화 버튼: 고객에게 바로 전화
